## 🛠 Project Overview
A comprehensive digital platform for **Santoshi Electro Engineering**, showcasing industrial-grade control panels and robotic systems. This project bridges the gap between traditional engineering and modern web technology.

### ✨ Features
* **Industrial Showcase:** Dynamic product pages for Robotic Handling & Welding.
* **Asset Management:** Integrated PDF technical manuals.
* **Client Portfolio:** Grid-based display of industry partners.
* **Responsive UI:** Seamless experience across all devices.

## 💻 Tech Stack
- **Framework:** React.js
- **Styling:** Custom CSS3 & Modules
- **Data:** Modular JSON-ready components
- **Assets:** Optimized PNG/SVG for industrial graphics
